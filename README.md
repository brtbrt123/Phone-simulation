# 📱 Smartphone OOP Simulation (Java)

This project is a simplified simulation of how a smartphone works, developed using **Object-Oriented Programming in Java**.  
The program demonstrates how different parts of a smartphone interact while performing common daily activities such as sending messages, playing music, taking photos, and browsing the internet.

## 🚀 Features
- Simulates a smartphone device with multiple capabilities
- Includes different applications with unique behaviors
- Uses internal components like Battery, Camera, and Network
- Shows interaction between user, smartphone, apps, and components
- Displays messages to show how the system works step-by-step

## 🧠 OOP Concepts Used
- **Inheritance** – Apps inherit from a base App class
- **Aggregation** – Smartphone contains components (Battery, Camera, Network)
- **Association** – User interacts with Smartphone

## ▶️ Simulation Flow
- Smartphone starts
- User sends a message
- User plays music
- User takes a photo
- Smartphone connects to internet
- User browses using browser app

## 🎓 Purpose
This project was created as an academic requirement to demonstrate proper
Object-Oriented Programming design, class relationships, and system simulation in Java.
